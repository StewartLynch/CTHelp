## About CTHelp

CTHelp is a Help system for iOS/Swift 4.2+

![SampleScreen](SampleScreen.gif)

## Requirements

Xcode 10+, Swift 4.2+

## Installation

CTHelp is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'CTHelp'
```
